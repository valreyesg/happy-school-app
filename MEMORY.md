# Happy School App — Memoria del Proyecto

## Sobre el proyecto
- App para escuela privada pequeña llamada Happy School
- Slogan: Comunidad Infantil
- Colores: rojo, amarillo, verde y morado
- ~25 alumnos, 5 grupos: Maternal, Prekinder, K1, K2, K3
- 4 maestras, 1 maestra de inglés, 1 directora, 1 administrativo

## Sobre la dueña del proyecto
- Nombre: Valeria
- Rol: mamá de alumnas y cliente del proyecto
- Prefiere explicaciones claras y en español
- El desarrollo debe ser opensource en GitHub

## Reglas que NUNCA deben cambiar
- Todo el sistema en español
- Máximo 2 toques para acciones frecuentes
- Nunca hardcodear catálogos en el código
- Siempre leer MEMORY.md, CONTEXT.md y PENDIENTES.md al iniciar sesión
- Siempre actualizar CONTEXT.md y PENDIENTES.md al terminar sesión
- Diseño infantil y colorido en todo momento
- No tomar decisiones de arquitectura sin consultar CONTEXT.md
- Nunca cambiar el stack sin autorización de Valeria

## Stack tecnológico fijo
- React Native + Expo (móvil)
- React + Tailwind (web)
- Node.js + Express + PostgreSQL
- JWT autenticación
- Cloudinary (fotos y documentos)
- Firebase Cloud Messaging
- WhatsApp Business API
- Google Calendar API

## Frase para iniciar cada sesión
"Lee MEMORY.md, CONTEXT.md y PENDIENTES.md en ese orden y dime en qué quedamos antes de continuar."
